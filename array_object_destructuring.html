Array destructuring

Array destructuring in JavaScript is a convenient way to extract values from arrays and assign them to variables in a single, concise statement.

üîπ Basic Syntax:

const array = [1, 2, 3];
const [a, b, c] = array;

console.log(a); // 1
console.log(b); // 2
console.log(c); // 3

üîπ Skip Elements:

You can skip elements using commas:


const [first, , third] = [10, 20, 30];
console.log(first); // 10
console.log(third); // 30

üîπ Default Values:

You can set default values if the item is undefined:

const [a = 1, b = 2, c = 3] = [10];
console.log(a); // 10
console.log(b); // 2 (default)
console.log(c); // 3 (default)

üîπ Swapping Variables:

A common trick:

let x = 1, y = 2;
[x, y] = [y, x];
console.log(x); // 2
console.log(y); // 1

üîπ Rest Syntax:

You can collect the remaining elements:

const [first, ...rest] = [1, 2, 3, 4];
console.log(first); // 1
console.log(rest); // [2, 3, 4]

üîπ Nested Destructuring:

const [a, [b, c]] = [1, [2, 3]];
console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
Object destructuring

Object destructuring lets you extract properties from an object and assign them to variables in a clean, readable way.

‚úÖ Basic Syntax:

const person = { name: "Alice", age: 30 };

const { name, age } = person;

console.log(name); // "Alice"
console.log(age);  // 30

‚úÖ Renaming Variables:

You can rename variables while destructuring:

const user = { id: 101, username: "Mark" };

const { id: userId, username: name } = user;

console.log(userId); // 101
console.log(name);   // "Mark"

‚úÖ Default Values:

Use default values for properties that might be undefined: 

const { a = 1, b = 2 } = { a: 10 };

console.log(a); // 10
console.log(b); // 2 (default value)

‚úÖ Nested Destructuring:

You can extract values from nested objects:

const person = {
  name: "Tom",
  address: {
    city: "London",
    zip: "E1 6AN"
  }
};

const { address: { city, zip } } = person;

console.log(city); // "London"
console.log(zip);  // "E1 6AN"

‚úÖ Destructuring in Function Parameters:

You can destructure directly in function arguments:

function greet({ name, age }) {
  console.log(`Hello ${name}, age ${age}`);
}

const user = { name: "Lena", age: 28 };
greet(user); // Hello Lena, age 28

‚úÖ Rest Syntax:

Use the rest operator to gather the remaining properties:

const user = { id: 1, name: "Eva", age: 25 };

const { name, ...rest } = user;

console.log(name); // "Eva"
console.log(rest); // { id: 1, age: 25 }
