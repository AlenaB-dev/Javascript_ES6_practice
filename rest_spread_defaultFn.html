Rest

The rest operator in JavaScript is written as three dots: .... It allows to collect multiple elements into a single array or object. It's mostly used in function parameters and in destructuring.

üîπ Syntax

function myFunction(...args) {
  console.log(args);
}

Here, args becomes an array containing all the arguments passed to myFunction.

‚úÖ Common Uses

1. In Function Parameters.
To collect remaining arguments into an array: 

function sum(...numbers) {
  return numbers.reduce((total, num) => total + num, 0);
}

console.log(sum(1, 2, 3)); // 6

2. In Array Destructuring.To gather the "rest" of the items: 

const [first, second, ...others] = [10, 20, 30, 40];
console.log(first);  // 10
console.log(second); // 20
console.log(others); // [30, 40]

3. In Object Destructuring.To gather remaining properties: 

const user = { name: "Alice", age: 30, city: "London" };
const { name, ...rest } = user;

console.log(name); // Alice
console.log(rest); // { age: 30, city: "London" }

‚ö†Ô∏è Difference from Spread Operator

  • Rest gathers into a collection (e.g., ...args)
  • Spread spreads out elements (e.g., myFunc(...array))

They look the same (...) but behave oppositely, depending on context.

Spread

üîπ Syntax

Spread operator in JavaScript ‚Äî it's also written as ..., just like the rest operator, but it does the opposite: instead of gathering items into one, it spreads them out.

It allows an iterable (like an array or string) or an object to be expanded in places where multiple elements or key-value pairs are expected.

‚úÖ Common Uses

1. Copying Arrays

const arr1 = [1, 2, 3];
const arr2 = [...arr1]; // makes a shallow copy

console.log(arr2); // [1, 2, 3]

2. Combining Arrays

const a = [1, 2];
const b = [3, 4];
const combined = [...a, ...b];

console.log(combined); // [1, 2, 3, 4]

3. Spreading in Function Calls

function add(x, y, z) {
  return x + y + z;
}

const nums = [1, 2, 3];
console.log(add(...nums)); // 6

4. Copying or Merging Objects

const user = { name: "Alice", age: 25 };
const updatedUser = { ...user, city: "London" };

console.log(updatedUser);
// { name: 'Alice', age: 25, city: 'London' }

5. Spreading Strings into Characters

const word = "Hi!";
const letters = [...word];

console.log(letters); // ["H", "i", "!"]

üëâ Spread is used when giving values. Rest is used when receiving them.
