forEach()

The forEach() method is one of the most commonly used array helper methods in JavaScript. It's used to iterate over elements in an array and execute a function for each element.

✅ Use forEach() when:

  • You need to do something with each item (side effects)
  • You don’t need to return or build a new array

🔹 Syntax

array.forEach((element, index, array) => {
  // your code here
});
  • element: The current item in the array
  • index (optional): The index of the current item
  • array (optional): The array being iterated over

🔹 Example

const fruits = ['apple', 'banana', 'cherry'];

fruits.forEach((fruit, index) => {
  console.log(`${index}: ${fruit}`);
});

Output:
0: apple
1: banana
2: cherry

🔹 Key Features

  • Does not return a new array (unlike map(), filter())
  • Does not stop early (unlike a for loop with break)
  • Mostly used for side effects (e.g., logging, updating the DOM, etc.)

🔹 Real-World Use Case

const users = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 },
];

users.forEach(user => {
  console.log(`${user.name} is ${user.age} years old.`);
});

🔹 Common Mistakes

❌ Trying to return values: forEach() doesn’t return anything useful — use map() if you want to create a new array. 

const result = fruits.forEach(fruit => {
  return fruit.toUpperCase();
});

console.log(result); // undefined
map()

The map() method is used to create a new array by applying a function to each element of an existing array. It does not modify the original array.

🔹 Syntax

const newArray = array.map((element, index, array) => {
  // return transformed element
});
  • element — the current item in the array
  • index (optional) — the index of the current item
  • array(optional) — the array map() was called on

🔹 Example

const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2);

console.log(doubled); // [2, 4, 6]

🔹 Key Features

  • Returns a new array with the same number of elements.
  • Does not change the original array.
  • Commonly used to transform or extract data.

🔹 Real-World Use Case

const users = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 }
];

const names = users.map(user => user.name);
console.log(names); // ["Alice", "Bob"]

🔹 Common Mistakes

Forgetting to use return inside the callback when using a function block ({}). 

// Incorrect:
numbers.map(function(num) {
  num * 2; // No return, so it returns undefined
});

// Correct:
numbers.map(function(num) {
  return num * 2;
});
filter()

The filter() method creates a new array containing only the elements that pass a test (a function you provide). It does not modify the original array.

🔹 Syntax

array.filter(callback(element, index, array), thisArg)
  • callback: A function that returns true to keep the element, or false to skip it.
  • element: The current element being processed.
  • index (optional): Index of the current element.
  • array (optional): The array being processed.
  • thisArg (optional): Value to use as this inside the callback.

🔹 Example 1: Filter Numbers Greater Than 10

const numbers = [5, 12, 8, 20, 3];

const filtered = numbers.filter(num => num > 10);

console.log(filtered); // [12, 20]

🔹 Example 2: Filter Even Numbers

const numbers = [1, 2, 3, 4, 5, 6];

const evens = numbers.filter(num => num % 2 === 0);

console.log(evens); // [2, 4, 6]

🔹 Example 3: Filter Objects by Property

const users = [
  { name: "Alice", isActive: true },
  { name: "Bob", isActive: false },
  { name: "Charlie", isActive: true }
];

const activeUsers = users.filter(user => user.isActive);

console.log(activeUsers);
/*
[
  { name: "Alice", isActive: true },
  { name: "Charlie", isActive: true }
]
*/
find()

The find() method returns the first element in the array that satisfies the provided testing function. If no element matches, it returns undefined.
It does not modify the original array.

🔹 Syntax

array.find(callback(element, index, array), thisArg)
  • callback: A function that is called on each element.
  • element: The current element being processed.
  • index (optional): The index of the current element.
  • array (optional): The array being processed.
  • thisArg (optional): Value to use as this inside the callback.

🔹 Example 1: Find First Number Greater Than 10

const numbers = [3, 8, 12, 5, 20];

const found = numbers.find(num => num > 10);

console.log(found); // 12

🔹 Example 2: Find Object by Property

const users = [
  { id: 1, name: "Alice" },
  { id: 2, name: "Bob" },
  { id: 3, name: "Charlie" }
];

const user = users.find(u => u.id === 2);

console.log(user); // { id: 2, name: "Bob" }

⚠️ Note:

  • find() stops searching as soon as it finds a match.
  • If no match is found, it returns undefined.
every()

The every() method in JavaScript is an array helper that checks if all elements in an array satisfy a condition.

🔹 Syntax

array.every(callback(element, index, array))
  • callback is a function that runs on each element.
  • Returns true if every element passes the test.
  • Returns false as soon as one fails.

🔹 Example 1: All numbers positive

const numbers = [2, 4, 6, 8];
const allPositive = numbers.every(num => num > 0);
console.log(allPositive); // true

🔹 Example 2: Not all conditions met

const numbers = [2, -1, 6];

const allPositive = numbers.every(num => num > 0);
console.log(allPositive); // false (because -1 is not > 0)

🔹 Example 3: Strings of certain length

const words = ["apple", "banana", "cherry"];

const allLongEnough = words.every(word => word.length >= 5);
console.log(allLongEnough); // true

🔹 Common mistake:

const result = [1, 2, 3].every(x => {
  x > 0; // returns undefined!
});
console.log(result); // false

You must return the condition:

const result = [1, 2, 3].every(x => x > 0); // ✅ Correct
some()

Let’s look at the some() array helper method — it’s the opposite of .every()in many ways.

✅ What .some() does:

  • some() checks if at least one element in an array passes a condition.

🔹 Syntax

array.some(callback(element, index, array))
  • Returns true if any element satisfies the test.
  • Returns false only if none of them do.

🔹 Example 1: At least one negative number

const numbers = [1, 2, -3, 4];

const hasNegative = numbers.some(num => num < 0);
console.log(hasNegative); // true

🔹 Example 2: No matches

const numbers = [2, 4, 6];

const hasOdd = numbers.some(num => num % 2 !== 0);
console.log(hasOdd); // false

🔹 Example 3: Strings in array

const fruits = ["apple", "banana", "cherry"];

const hasShortName = fruits.some(fruit => fruit.length < 5);
console.log(hasShortName); // true ("apple" has 5, but still counts as short here)

⚖️ Comparison: every() vs some()

Method Description Returns true if...
every() All elements match condition Every element passes the test
some() At least one element matches One or more elements pass the test

🔁 Tip:

You can combine both to write clear logic:

const users = [
  { name: "Anna", age: 30 },
  { name: "Bob", age: 17 }
];

const hasUnderage = users.some(user => user.age < 18); // true
const allAdults = users.every(user => user.age >= 18); // false
include()

The includes() method is a simple and powerful JavaScript array helper that checks whether a certain element exists in an array.

🔹 Syntax

array.includes(valueToFind, fromIndex);
  • valueToFind — the value you want to search for.
  • fromIndex (optional) — the index to start the search from (default is 0).

🔹 Example 1: Basic usage

const fruits = ["apple", "banana", "orange"];

console.log(fruits.includes("banana")); // true
console.log(fruits.includes("grape"));  // false

🔹 Example 2: Starting from a specific index

const numbers = [1, 2, 3, 4, 2];

console.log(numbers.includes(2));       // true
console.log(numbers.includes(2, 3));    // true (searches from index 3)
console.log(numbers.includes(2, 4));    // true (2 is at index 4)
console.log(numbers.includes(2, 5));    // false (starts after end)

⚠️ Notes:

  • includes() uses strict equality (===) when comparing.
  • It doesn’t work well with searching inside objects or arrays of objects directly (you’d use some() or find() for that).

🔹 Example 3: Doesn’t work for deep equality

const users = [{ name: "Alena" }, { name: "Anna" }];

console.log(users.includes({ name: "Alena" })); // false (different object reference)

To find objects inside an array, you can use some():

console.log(users.some(user => user.name === "Alena")); // true
